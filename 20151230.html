二零一五(结语)

人生

就像《伪装者》里，你以为“除了你自己，谁也不能相信” 已经成了你人生的座右铭； 后来你又发觉 “只有信任别人，你才能活到现在”。也许以后，你又会有新的认知。 无论任何，对于事物抱持陈腐不变的态度，显然不是这个时代大智大慧之人该有的态度。

知道了喜怒，才了解好恶;体验了哀乐，也就明白了期待。一直在不停的往前跑，忽然停下来回头看的时候， 发现后面一片空白，莫名的愈发孤独。

朋友

而今年最重要的事，显然是我重新认识了一个朋友，这对我至关重要。

技能

从Sencha touch 到app framework ,从phonegap 到apiCloud。 尝试了很多，各有优劣。说到坑，天下软件皆有坑，别人挖的自己填，坑大坑小都是选择。 如果只是想快速开发一个功能简单的移动app ，Sencha touch 倒是可以作为备选方案。 如果希望在界面表现，功能扩展上表现出色，app framework 虽然有坑，也是可以作为备选方案的。 phonegap坑超乎想象的多，它的复杂在于与native code 的接合配置复杂，代码控件可选方案很少， 第三方控件良莠不齐。 而中间件表现上apiCloud虽然偶尔不稳定，但整体表现良好。复杂在第三方平台组件接口逻辑复杂， 因为有良好的接口文档，使用起来很方便，但掩盖不了IDE的问题。配置项很多，但上手快。 与融云，极光，微信的等第三方平台的集成较快。 而如果是对用户体验有较高要求，html5 app与原生native还是差距较大。

事业

我自己

从技能领域上来说，我算是又上了一个新的台阶；从职业规划来说，我却是刻意逃避了一些事情，反倒是倒退了。 从身体状态来说，今年的损失有点大，从认知视野来说，我多少有些收获。从人生态度来说，我有些态度更加坚定了， 有些态度却愈发看淡了。得得失失，有得必有失，然有失才能有得。
